2) Uma maquina virtual é um software que permite instalar programas e executa-los sem afetar o sistema operacional principal.

1-	Instalação e operação simples
2-	Isolamento de dados e bugs
3-	Compatibilidade
Desvantagens:
1-	Alto consumo de memória RAM e HD
2-	Para um bom desempenho é necessário um bom hardware

3) Objeto é definido neste modelo como um conceito, abstração ou coisa com limites e significados bem definidos para a aplicação em questão. Já uma classe de objetos descreve um grupo de objetos com propriedades (atributos) similares, comportamento (operações) similares, relacionamentos comuns com outros objetos e uma semântica comum.

4) Metodos – São as coisas(funções) que um determinado objeto executa ou pode executar.
Atributos - São as características de objetos.

5) Public – Metodos, atributos ou classes que são visiveis e modificaveis para qual todos tem acesso;
Protected – Se uma classe (mesmo que esteja fora do pacote) estende da classe com o atributo protected, ela terá acesso a ele.
Private – A única classe que tem acesso ao atributo é a própria classe que o define.
Default –  Todas as classes que estiverem no mesmo pacote que a classe que tem acesso.
